# Kleiber: Organ-Based Reconstruction of Human Metabolic Scaling

This repository contains the code used for the analyses presented in:

Thomas DM, Müller M, Bosy-Westphal A, Heymsfield SB. *The organ basis of human metabolic scaling.*

## Repository Contents

### Clean Females Notebook.ipynb

Analysis of the female cohort (n = 173).

This notebook:

* Fits organ-specific power-law scaling relationships
* Estimates organ-specific scaling exponents and confidence intervals
* Performs diagnostic analyses
* Calculates organ metabolic weighting coefficients
* Computes effective whole-body scaling exponents
* Generates summary statistics and tables

### Clean Males Notebook.ipynb

Analysis of the male cohort (n = 131).

This notebook:

* Fits organ-specific power-law scaling relationships
* Estimates organ-specific scaling exponents and confidence intervals
* Performs diagnostic analyses
* Calculates organ metabolic weighting coefficients
* Computes effective whole-body scaling exponents
* Generates summary statistics and tables

## Data Availability

The underlying participant-level body composition and metabolic data are not included in this repository because of participant-consent limitations and applicable data-protection regulations.

## Software

Analyses were performed in Python 3.11 using:

* pandas
* numpy
* scipy
* statsmodels
* matplotlib
